### Pre-requisites for owasp_dependency_checker:
Install the dependency-check tool before running the script. You can install the dependency-check tool using the following command: brew install dependency-check.

### Pre-requisites for packj.py:
Install the packj library and the jq tool before running the script. You can install the packj library using the following command: pip install packj. The jq tool can be installed using the package manager for your operating system.
